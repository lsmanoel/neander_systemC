# Neander - SystemC

* **to Clear:**
		$ rm -rf build

* **to Build:**
		$ mkdir build && cd build && cmake ../ && make && ./neander && cd ..
	
* **to ReBuild:**
		$ rm -rf build && mkdir build && cd build && cmake ../ && make && ./neander && cd ..
